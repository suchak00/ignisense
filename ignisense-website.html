About IGNISENSE

IGNISENSE is an IoT-based natural disaster monitoring system designed to detect fire risks using environmental sensors such as temperature, humidity, soil moisture, and smoke sensors. The data is collected using ESP8266 NodeMCU and sent to the cloud using ThingSpeak for real-time monitoring.
