About IGNISENSE

IGNISENSE is an advanced IoT-based natural disaster monitoring system designed to detect fire hazards and environmental risks using real-time sensor data. The system integrates temperature, humidity, soil moisture, and flame sensors to provide early warning alerts.

Data is collected using ESP32 and transmitted to the cloud via ThingSpeak for live monitoring and analysis.
